# Zone Bridge - 蓝黄区跨网络 Agent 通信工具

## 问题场景

两个网络环境：
- **蓝区 (Blue Zone)**: 外部网络，无限制
- **黄区 (Yellow Zone)**: 公司内部网络，通过 Web 代理/网关访问蓝区

### 网络约束

| 方向 | 连通性 | 限制 |
|------|--------|------|
| 黄区 → 蓝区 | ✅ 无限制 | - |
| 蓝区 → 黄区 | ⚠️ 受限 | 内容 < 50KB |

## 项目目标

让蓝区和黄区能够互相访问各自的 Agent (A2A 或自定义协议)

## 实验计划

### 实验 1: 黄区 → 蓝区
- 在蓝区启动 HTTP server
- 从黄区 curl 测试连通性

### 实验 2: 蓝区 → 黄区  
- 在黄区启动 HTTP server
- 从蓝区 curl 测试连通性和 50KB 限制
